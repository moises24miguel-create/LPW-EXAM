# Web Development Projects Collection

This repository contains a collection of web development projects created using HTML5, CSS3, JavaScript, jQuery, PHP, MySQL, AJAX, and Bootstrap 5.  
The projects demonstrate frontend development, backend integration, database operations, responsive design, and dynamic web functionalities.

---

# Projects Included

## 1. Static Webpage with HTML & CSS
A responsive multi-section webpage demonstrating:
- Semantic HTML5
- Tables & Lists
- Forms & Media
- Custom CSS Styling
- Responsive Layout

---

## 2. PHP CRUD with MySQL
A basic CRUD application using PHP & MySQL:
- Insert Records
- Display Data
- Edit & Delete Operations
- Database Connectivity
- PHP Form Handling

---

## 3. jQuery Selectors & AJAX Request
Frontend project demonstrating:
- jQuery Selectors
- DOM Traversal
- Event Handling
- AJAX Requests
- Dynamic Content Updates

## 4. Full-Stack Student Record System
A complete student management system featuring:
- Admin Authentication
- Student CRUD Operations
- AJAX-based Search & Delete
- File Uploads
- Bootstrap Responsive UI
- Client-side & Server-side Validation
- jQuery Effects & Dynamic Table Rendering
